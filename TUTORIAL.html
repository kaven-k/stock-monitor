📋 项目概述

StockMonitor 是一套面向 A 股短线交易者的实时监控与 AI 选股系统，核心能力：

  • 实时行情监控 — 腾讯财经数据源，不封 IP，5 秒刷新
  • K 线 & 技术指标 — 日/周/月 K，MA/MACD/RSI/BOLL/KDJ 全量计算
  • 多维度预警 — 13 种预警规则（价格突破/涨跌幅/量比/涨跌停等），支持飞书通知
  • AI 智能选股 — DeepSeek 大模型分析全市场，6 维框架选股，自动记录供复盘
  • 市场概览 — 三大指数 + 恐慌贪婪指数 + 板块排名 + 资金流向 + 主线板块识别
  • 深色 / 浅色主题 — 一键切换，ECharts 图表自适应

⚙️ 配置

在项目根目录 stock-monitor/ 下创建 .env 文件：

# ---------- 必填：DeepSeek API Key ----------
# 注册地址：https://platform.deepseek.com
DEEPSEEK_API_KEY=sk-your-key-here

# ---------- DeepSeek 模型 ----------
DEEPSEEK_MODEL=deepseek-v4-pro

# ---------- 飞书通知（可选，不需要可留空）----------
FEISHU_APP_ID=
FEISHU_APP_SECRET=
FEISHU_USER_ID=

# ---------- 服务端口（默认 5000）----------
API_PORT=5000
💡 没有 DeepSeek Key 也能用！
不配 API Key 时，行情监控、K 线、预警等功能完全正常，只是不能使用 AI 一键选股和对话选股。

📖 功能详解

1. 添加监控股票

点击顶部栏 + 添加股票，搜索股票代码或名称，支持模糊搜索。添加后右侧详情面板可查看 K 线图和技术指标。

2. K 线图 & 技术指标

点击任意股票行打开详情面板：

  • K 线 — 日/周/月切换，支持缩放、拖拽
  • 技术指标 — MACD / RSI / KDJ / BOLL，纯 Python 计算
  • 基本信息 — 价格、涨跌幅、换手率、PE/PB、市值

3. 预警规则

左侧 规则管理 或顶部 + 预警规则，支持 13 种预警类型：

类型触发条件
price_up股价向上突破指定价位
price_down股价向下跌破指定价位
change_up当日涨幅超过阈值
change_down当日跌幅超过阈值
volume_surge成交量突然放大
turnover_high换手率超过阈值
amplitude_high日内振幅过大
price_break_ma价格突破均线
continuous_up连续 N 日上涨
continuous_down连续 N 日下跌
volume_ratio量比异常
limit_up触及涨停
limit_down触及跌停

规则创建后，每 5 秒检查一次，触发时页面弹窗 + 可选飞书通知。

4. 股票分组

左侧 股票分组 区域，点击 + 创建分组（如"半导体""医药"），将股票拖入对应分组，便于按板块查看。

5. 市场概览

左侧 📊 市场概览 查看：

  • 三大指数（上证/深证/创业板）实时行情
  • 恐慌贪婪指数 + 市场涨跌比
  • 行业涨幅 TOP10 + 概念涨幅 TOP10
  • 板块资金流向（流入/流出 TOP5）
  • 主线板块识别（综合评分模型）

6. AI 智能选股 ⭐

左侧 🤖 智能选股，两种模式：

⚡ 一键选股 — 点击按钮，AI 自动从全市场筛选 8 只最佳短线标的，输出评分、买入逻辑、建议买点、止损价、目标价。结果自动保存到数据库。
💬 对话选股 — 自由输入条件（如 "推荐半导体板块的短线标的"），AI 结合实时市场数据给出分析和推荐。

7. AI 选股记录（复盘）

左侧 📋 AI选股记录：

  • 按日期分组展示所有历史 AI 推荐
  • 每条记录包含：买点、评分、推荐理由、止损/目标
  • 记录推荐时的行情快照（现价、涨跌幅、成交量）供复盘对比
  • 预留复盘标记字段（命中目标 / 触发止损 / 平盘）

8. 深色 / 浅色主题

顶部栏右侧点击 🌙 / ☀️ 切换，偏好自动保存到 localStorage。

9. 监控控制

顶部 ⏸ 暂停监控 可临时停止行情推送，刷新间隔可从下拉菜单调整（3/5/10/30 秒）。

🔧 常见问题

Q: 页面显示"暂无股票数据"
A: 服务器进程可能异常，重启 backend/app.py 即可。长时间运行的 Flask 开发服务器偶尔会进入异常状态。
Q: AI 选股报错"解析失败"
A: DeepSeek 偶发返回非标准 JSON。已内置 3 级降级解析策略，重试一次通常可解决。
Q: 周 K / 月 K 无法切换
A: 确认网络可访问东方财富 API。日线优先从数据库缓存读取，周月线走 API 实时拉取。
Q: 飞书通知收不到
A: 检查 .env 中飞书三要素是否正确。需要先在飞书开放平台创建应用并获取凭据。
Q: 前端改动后没有生效
A: 浏览器缓存了旧 JS 文件。刷新页面（Ctrl + Shift + R）或清除缓存即可。生产环境已配置 Cache-Control: no-cache。
