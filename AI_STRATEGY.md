# AI 智能选股策略说明书 v1.1

> 最后更新：2026-06-27  
> 适用模型：DeepSeek v4 Pro  
> 适用场景：A股超短线交易（持仓 ≤ 5天）

---

## 一、策略核心理念

### 1.1 用户画像
| 属性 | 设定 |
|------|------|
| 交易风格 | 超短线（追涨+轮动） |
| 最大持仓周期 | 5个交易日（周一买→周五必卖） |
| 止损规则 | -5% 无条件离场 |
| 选股偏好 | 强势板块中的启动期个股，不追涨停板 |
| 仓位管理 | 单票 ≤ 20%，总仓位 ≤ 80% |

### 1.2 核心矛盾
```
追涨 ≠ 追涨停
```
- **追涨**：在趋势确立后、仍有上涨空间时介入（涨幅 2%-7% 区间）
- **追涨停**：在股价已封板（涨 ≥ 9.8%）时企图买入——**封板后无法成交，且次日高开低走概率 > 60%**

## 二、六维分析框架

AI 选股从以下 6 个维度综合评估每只候选股：

### 2.1 技术面（权重 30%）
| 指标 | 理想状态 | 减分项 |
|------|----------|--------|
| 均线排列 | MA5 > MA10 > MA20（多头） | MA5 下穿 MA10 |
| MACD | DIF > DEA，红柱放大 | 死叉、绿柱放大 |
| 成交量 | 温和放量（量比 1.2-3.0） | 缩量或异常巨量（量比 > 5） |
| K线形态 | 阳线实体饱满，上影线短 | 长上影线、十字星 |
| RSI | 60-75（强势但未超买） | RSI > 85（超买风险） |

### 2.2 资金面（权重 25%）
- **主力净流入**：板块资金持续流入 > 3 天
- **大单占比**：大单买入占比 > 大单卖出占比
- **换手率**：3%-15% 为健康区间，< 1% 缺乏流动性，> 25% 异常
- **成交额**：日成交额 > 1 亿（排除僵尸股）

### 2.3 龙头效应（权重 15%）
- 优先选择板块**领涨龙头**（最早涨停/涨幅最大/封单最强）
- 若龙头已涨停无法买入 → 选**次龙头**（涨幅第二、跟风紧密）
- 龙头与跟风股涨幅差距应 < 3%

### 2.4 市场情绪（权重 10%）
- **恐慌贪婪指数**：> 60 表示市场偏乐观，适合操作；< 40 偏恐慌，降低仓位
- **涨跌比**：涨家数 > 跌家数时胜率更高
- **涨停家数**：50-100 家为健康区间，> 150 家过热警惕

### 2.5 国际行情（权重 5%）
- **美股映射**：隔夜纳指/标普涨跌对 A 股科技股有传导效应
- **美元/人民币**：人民币升值 → 外资流入利好
- **大宗商品**：原油/铜价影响相关板块

### 2.6 政策/消息面（权重 15%）
- 国务院/发改委/工信部最新政策方向
- 行业重大事件（新品发布、订单公告、业绩预增）
- 概念题材持续性判断（一日游 vs 主线行情）

## 三、硬性过滤规则（一票否决）

以下类型股票**绝对禁止推荐**，即使其他维度评分再高：

| 规则 | 阈值 | 原因 |
|------|------|------|
| 🚫 已涨停 | 涨幅 ≥ 9.8% | 封板无法买入，次日高开低走概率大 |
| 🚫 已跌停 | 跌幅 ≤ -9.8% | 弱势确认，次日大概率继续跌 |
| 🚫 ST/\*ST | 股票名称含 ST | 退市风险，流动性极差 |
| 🚫 新股/次新股 | 上市 < 60天 | 波动剧烈，无技术分析基础 |
| 🚫 日成交额 | < 5000万 | 流动性太差，进出困难 |
| 🚫 换手率 | > 30% | 异常换手，可能有庄家出货 |
| ⚠️ 接近涨停 | 涨幅 7%-9.5% | 需特别谨慎，若已封板则排除 |

## 四、评分模型

### 4.1 综合评分（满分 100）

```
总分 = 技术面(30) + 资金面(25) + 龙头效应(15) + 市场情绪(10) + 国际行情(5) + 政策面(15)
```

### 4.2 评分等级

| 分数 | 等级 | 操作建议 |
|------|------|----------|
| 85-100 | ⭐⭐⭐⭐⭐ 强烈推荐 | 可满仓介入 |
| 70-84 | ⭐⭐⭐⭐ 推荐 | 半仓介入 |
| 60-69 | ⭐⭐⭐ 谨慎推荐 | 轻仓试探 |
| < 60 | ⭐⭐ 不推荐 | 观望 |

## 五、推荐输出规范

### 5.1 一键选股模式
每次输出**严格 8 只**股票，包含以下字段：

```json
{
  "summary": "一句话总结当前市场环境",
  "stocks": [
    {
      "code": "600519",
      "name": "贵州茅台",
      "score": 85,
      "reason": "白酒板块资金持续流入第3日，茅台作为绝对龙头...",
      "hold_days": 3,
      "stop_loss": "1680.00",
      "target": "1780.00"
    }
  ]
}
```

### 5.2 推荐理由要求
- **字數**：80-150字
- **必须涵盖**：技术面 + 资金面 + 板块面 三个维度
- **必须说明**：为什么现在介入、预期空间多大、止损在哪里

## 附录：为什么涨停股不是好选择？

### 数据依据
- A 股历史回测：涨停次日高开概率约 65%，但**高开低走概率 > 60%**
- 涨停买入的实际困境：散户无法在涨停价成交，只能次日追高
- 次日开盘涨幅中位数约 3%，但收盘涨幅中位数仅 0.5%
- **结论**：涨停日的"浮盈"你拿不到，次日进入大概率亏损

### 替代策略
- 找涨幅 3%-6% 的**启动期**标的 → 当天还能买入，次日有继续上涨空间
- 关注**板块轮动**：A 板块涨完后资金流向 B 板块，提前埋伏
- **龙头买不到就买次龙头**：涨幅第二、逻辑类似的标的往往更安全
