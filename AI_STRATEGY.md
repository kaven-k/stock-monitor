> 最后更新：2026-07-15  
> 适用模型：DeepSeek v4 Pro (max_tokens=32768)  
> 适用场景：A股超短线交易（持仓 ≤ 5天）  
> 对应代码：`backend/ai_screener.py` + `backend/signal_engine.py`  
> ⚠️ 选股覆盖范围：全A股市场（东方财富全市场板块龙头 + 用户监控列表），按 `config.SCREENING_CONFIG` 全局过滤无交易权限板块。当前可用资金约 2 万元，已排除 **科创板(688xxx)**、**创业板(300/301xxx)** 与 **北交所(83/87/88/92/43xxxx)**（账户资产门槛分别为≥50万/≥10万/≥50万）；如已开通某板块权限，删除 `SCREENING_CONFIG["exclude_boards"]` 中对应行即可恢复。

## 一、策略核心理念

### 1.1 用户画像
| 属性 | 设定 |
|------|------|
| 交易风格 | 超短线（追涨+轮动） |
| 最大持仓周期 | 5个交易日（周一买→周五必卖） |
| 止损规则 | -5% 无条件离场 |
| 选股偏好 | 强势板块中的启动期个股，不盲目追涨停板 |
| 仓位管理 | 单票 ≤ 20%，总仓位 ≤ 80% |

### 1.2 核心原则
```
追涨 ≠ 追涨停 —— 关键看时机和概率
```
- **追涨**：在趋势确立后介入，无论涨幅大小，核心是判断次日是否还有溢价空间
- **追涨停（打板策略）**：如果封单够强、板块持续发酵，次日溢价空间可能很大；如果板块后劲不足、封单薄弱，风险极高
- **核心逻辑**：不是"能不能买涨停股"，而是"买了之后明天还有没有钱赚"——需要结合板块强度、资金面、市场情绪综合判断

## 二、六维分析框架

AI 选股从以下 6 个维度综合评估每只候选股：

### 2.1 技术面（权重 30%）
| 指标 | 理想状态 | 减分项 |
|------|----------|--------|
| 均线排列 | MA5 > MA10 > MA20（多头） | MA5 下穿 MA10 |
| MACD | DIF > DEA，红柱放大 | 死叉、绿柱放大 |
| 成交量 | 温和放量（量比 1.2-3.0） | 缩量或异常巨量（量比 > 5） |
| K线形态 | 阳线实体饱满，上影线短 | 长上影线、十字星 |
| RSI | 60-75（强势但未超买） | RSI > 85（超买风险） |

### 2.2 资金面（权重 25%）
- **主力净流入**：板块资金持续流入 > 3 天
- **大单占比**：大单买入占比 > 大单卖出占比
- **换手率**：3%-15% 为健康区间，< 1% 缺乏流动性，> 25% 异常
- **成交额**：日成交额 > 1 亿（排除僵尸股）

### 2.3 龙头效应（权重 15%）
- 优先选择板块**领涨龙头**（最早涨停/涨幅最大/封单最强）
- 若龙头已涨停无法买入 → 选**次龙头**（涨幅第二、跟风紧密）
- 龙头与跟风股涨幅差距应 < 3%

### 2.4 市场情绪（权重 10%）
- **恐慌贪婪指数**：> 60 表示市场偏乐观，适合操作；< 40 偏恐慌，降低仓位
- **涨跌比**：涨家数 > 跌家数时胜率更高（数据来源：全市场 5534 只A股统计）
- **涨停家数**：50-100 家为健康区间，> 150 家过热警惕

### 2.5 国际行情（权重 5%）
- **美股映射**：隔夜纳指/标普涨跌对 A 股科技股有传导效应
- **美元/人民币**：人民币升值 → 外资流入利好
- **大宗商品**：原油/铜价影响相关板块

### 2.6 政策/消息面（权重 15%）
- 国务院/发改委/工信部最新政策方向
- 行业重大事件（新品发布、订单公告、业绩预增）
- 概念题材持续性判断（一日游 vs 主线行情）

## 三、特殊状态评估指南

以下状态不代表直接淘汰，而是需要额外评估风险收益比：

| 状态 | 评估标准 | 何时可介入 | 何时应回避 |
|------|----------|-----------|-----------|
| 🚫 已涨停 | 封单强度、板块持续性 | 封单 > 流通市值5%、板块多股涨停共振、首板更安全 | 尾盘突袭拉板、跟风板、连板高位缩量加速 |
| 🚫 已跌停 | 是否为错杀/恐慌 | 基本面未恶化、板块无利空、大盘普跌拖累 | 业绩暴雷、行业政策利空、连续跌停 |
| ⚠️ 接近涨停 | 封板概率判断 | 量能配合良好、抛压小、板块同步拉升 | 无量拉升、板块独涨、多次触板回落 |
| ⚠️ 接近跌停 | 支撑位力度 | 关键均线支撑企稳、有大单托盘 | 跌破关键支撑、放量加速下跌 |
| ST/\*ST | 退市风险 | — | 始终回避（流动性差、退市风险高） |
| 新股/次新股 | < 60天 | — | 始终回避（无技术分析基础、波动剧烈） |

## 四、评分模型

### 4.1 综合评分（满分 100）

```
总分 = 技术面(30) + 资金面(25) + 龙头效应(15) + 市场情绪(10) + 国际行情(5) + 政策面(15)
```

### 4.2 评分等级

| 分数 | 等级 | 操作建议 |
|------|------|----------|
| 85-100 | ⭐⭐⭐⭐⭐ 强烈推荐 | 可满仓介入 |
| 70-84 | ⭐⭐⭐⭐ 推荐 | 半仓介入 |
| 60-69 | ⭐⭐⭐ 谨慎推荐 | 轻仓试探 |
| < 60 | ⭐⭐ 不推荐 | 观望 |

## 五、推荐输出规范

### 5.1 一键选股模式
每次输出**严格 8 只**股票，包含以下字段：

```json
{
  "summary": "一句话总结当前市场环境",
  "stocks": [
    {
      "code": "600519",
      "name": "贵州茅台",
      "score": 85,
      "reason": "白酒板块资金持续流入第3日，茅台作为绝对龙头...",
      "entry_price": "1750.00",
      "hold_days": 3,
      "stop_loss": "1680.00",
      "target": "1830.00"
    }
  ]
}
```

### 5.2 字段说明
| 字段 | 类型 | 说明 |
|------|------|------|
| code | string | 6位股票代码 |
| name | string | 股票简称 |
| score | int | 综合评分 0-100 |
| reason | string | 80-150字推荐理由，涵盖技术面+资金面+板块面 |
| **entry_price** | string | **买点价格**，AI 根据当日走势判断的最佳入场价位 |
| hold_days | int | 建议持有天数 (1-5) |
| stop_loss | string | 止损价位 |
| target | string | 目标价位 |

### 5.3 推荐理由要求
- **字数**：80-150字
- **必须涵盖**：技术面 + 资金面 + 板块面 三个维度
- **必须说明**：为什么现在介入、预期空间多大、止损在哪里
- **entry_price 必须给出**：结合当前价、均线支撑、成交量等因素推算合理买点

## 六、风控规则

| 规则 | 参数 |
|------|------|
| 单票止损线 | -5%（触发即卖，不犹豫） |
| 单票止盈线 | +8% ~ +15%（视强度而定） |
| 总仓位上限 | 80% |
| 单票仓位上限 | 20% |
| 连续止损 | 连亏 3 笔 → 当日停止交易 |
| 大盘风险 | 上证跌幅 > 2% → 减仓至 30% |

## 七、数据来源

| 数据类型 | 来源 | 备注 |
|----------|------|------|
| 实时行情 | 腾讯财经 HTTP API | 延迟约 3-5 秒 |

| AI 推理 | DeepSeek v4 Pro | 综合多维分析 |

## 八、系统增强功能 (v3.0 新增)

### 8.1 多指标共振信号引擎
- 文件：`backend/signal_engine.py`
- 基于 MA/MACD/RSI/BOLL/VWMA/ATR 加权评分（-100 ~ +100）
- 自动识别强买入/买入/中性/卖出/强卖出信号
- WebSocket 实时推送信号变化（`signal_update` 事件）

### 8.2 复合预警规则
- 文件：`backend/alert_engine.py`
- 支持 AND/OR 组合 11 种子规则类型
- 可针对单股或批量股票创建复杂触发条件

### 8.3 AI选股自动落库与复盘
- 每次选股自动保存到 `ai_picks` 表（含推荐时行情快照）
- 按时间戳分组（`YYYY-MM-DD HH:MM:SS`），多次选股不覆盖
- 自动创建同名分组（`🤖 AI 2026/07/01 14:30:25`），方便侧边栏查看实时涨跌
- 前端「AI选股记录」页面支持按日期→会话两级浏览

### 8.4 全市场情绪数据
- 东方财富 push2 API 遍历全部 ~5500 只A股
- 5 线程并行分页抓取 + 5 分钟缓存
- 前端「市场概览」面板显示实时情绪温度计
